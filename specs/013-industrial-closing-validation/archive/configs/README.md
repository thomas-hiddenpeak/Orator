# Archived Spec 013 Configurations

These TOML files record non-production speaker and rejected Sortformer
experiments from the Spec 013 investigation. They are not deployment profiles,
are not registered by CMake/CTest, and must not be used for new acceptance runs.

The repository-root `orator.toml` is the sole production speaker configuration.
The two non-archived `sortformer-v21-high-latency.toml` and
`sortformer-v21-low-latency.toml` files remain active only as numerical-oracle
fixtures for the same v2.1 checkpoint; they are not product configurations.

Historical reports may show the files at their original paths because those
commands describe past runs. Git history provides the exact historical layout.
